Clustering MBCC 2023


Perhaps this is the job that I have enjoyed the most because I brought together my two passions, craft beer and data.

The goal of this work is implement the clustering of the 786 different beers (and other drinks) from the MBCC 2023 made by Mikkeller, the famous Danish brewery.
The clustering begins with an exploratory data analysis, then apply the K-means algorithm with The Elbow Method and The Silhouettes to define the optimal number of clusters.

How I got the data?
The beer data of the 4 MBCC sessions has been collected from Untappd, using Power BI consolidated all data in a single csv file.

Available is the csv file with the raw data, all the Python code and the csv with the clustering result.

Clustered data





After the exploratory data analysis, the clustering was defined as follows, the cluster 0 for light beers and those below rating 4, such as Pilsners and Lagers. Cluster 1 for alcoholic beers such as Imperial stouts, Barleywines and Meads. Cluster 2 mostly for Wild ales, Lambics, Belgians, Farmhouse. For Sours cluster 3, except one that has 13.3 % Alc that falls in cluster 1. Finally cluster 4 for Ipas that has averages ratings.

Of course there may be other clustering, is possible to change the seed and run the algorithm several more times to find any other, for example perhaps at the request of the organization, they want to classify into 3 clusters, it is possible to try running the algorithm to see if there is any possible classification, although it is likely to be less optimal than the solution presented.
